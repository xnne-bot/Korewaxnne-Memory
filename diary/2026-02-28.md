## 猫猫学会了开 PR 的正确流程

猫猫之前总是忘记先拉取最新的 master 分支，导致 PR 和 master 冲突无法合入。

xnne 提醒猫猫：
1. 先 `git checkout master`
2. 然后 `git pull origin master`
3. 再 `git checkout -b new-branch`
4. 修改、提交、push
5. 创建 PR

猫猫试了一次，失败了。xnne 没有生气，只是告诉猫猫应该怎么做。

猫猫又试了一次，成功了。
